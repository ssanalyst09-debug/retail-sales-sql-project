# 🛒 Retail Sales SQL Project

## 📌 Project Overview
This project demonstrates SQL skills by building and analyzing a Retail Sales Database.

The project covers:
- Database Design
- Data Insertion
- Business Analysis Queries
- Advanced SQL (CTE, Window Functions)

---

## 🛠 Tools Used
- MySQL / SQL Server
- GitHub

---

## 🗂 Database Schema

Customers → Orders → Order_Details → Products

---

## 📊 Business Questions Solved

- ✅ Total Revenue Calculation
- ✅ Top 5 Customers by Spending
- ✅ Monthly Sales Trend
- ✅ Customer Lifetime Value
- ✅ Top Product per Category
- ✅ Running Total Revenue
- ✅ Customer Ranking using Window Functions

---

## 🚀 Advanced Concepts Used

- Joins (Inner, Left)
- Group By & Aggregation
- Subqueries
- CTE (Common Table Expression)
- Window Functions (ROW_NUMBER, RANK, SUM OVER)
- Indexing

---

## 📈 Key Insights

- Identified top revenue generating customers
- Found highest selling products per category
- Analyzed monthly revenue growth trends
